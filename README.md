### 写在前面
这是一个简单的后台管理模板后台管理系统一般使用较多有表格，图表，以及树状图。在这个简单的项目中我的使用node+express+mysql写了几个接口实现了部分功能，有登录，新增笔记，删除笔记，分页以及查看编辑笔记。前端使用技术栈有：vue-cli+vue+vuex+vue-router+axios+element-ui+echart
### 安装
npm i
### 运行 
npm run dev
### 访问地址
http://localhost:8080
### 打包
npm run build
